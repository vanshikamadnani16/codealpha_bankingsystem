# Banking System in C++

A simple console-based Banking System developed using **C++** and **Object-Oriented Programming (OOP)** concepts.

This project allows users to create bank accounts, deposit and withdraw money, transfer funds, and view transaction history.

---

# Features

- Create Customer Account
- Deposit Money
- Withdraw Money
- Transfer Funds
- Display Account Details
- Display Transaction History
- Menu-Driven Interface

---

# OOP Concepts Used

- Classes and Objects
- Constructors
- Encapsulation
- Member Functions
- Vectors (STL)
- Object Composition

---

# Classes Used

## 1. Transaction Class
Stores transaction details such as:
- Transaction Type
- Amount

## 2. Account Class
Manages:
- Account Number
- Balance
- Deposit & Withdrawal
- Transfer Operations
- Transaction History

## 3. Customer Class
Stores:
- Customer Name
- Customer ID
- Bank Account Details
